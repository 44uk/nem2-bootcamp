# NEM財団によるテストネット(MIJIN_TEST)

NEM財団関係者のノードによるネットワークです。

現時点では`MIJIN_TEST`ネットワークですが、将来的には`TESTNET`が公開されると思われます。

`TESTNET`が公開されれば有志によるノードが参加した大きなネットワークが形成されるものと思われます。
